# Machine Guide

This guide explains the parts of the simulator that make the computer usable as a small programmable machine.

The goal is to give a beginner-friendly path through the display, keyboard, binary loader, assembler, dogfood programs, and reusable assembly routines. It focuses on how these pieces fit into the machine that already exists in the repository.
